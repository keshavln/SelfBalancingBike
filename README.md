# Self Balancing Bike

## Overview
This repository contains the source code and CAD files for a bike that maintains an upright position using a reaction wheel. An LQR controller was chosen owing to its ability to simultaneously stabilize the bike and keep the reaction wheel's angular speed within the operating range of the motor. A demo can be viewed below.

## Usage
```
# Clone the repository:

git clone https://github.com/keshavln/SelfBalancingBike.git
cd SelfBalancingBike

# Install requirements:

pip install -r requirements.txt

# Run the simulation:

python sim.py
```
