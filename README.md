# Self Balancing Bike

## Overview
This repository contains the source code and CAD files for a bike that maintains an upright position using a reaction wheel and an LQR controller. A demo can be viewed below.

## Usage
```
# Clone the repository:

git clone https://github.com/keshavln/SelfBalancingBike.git
cd SelfBalancingBike

# Install requirements:

pip install -r requirements.txt

# Run the simulation:

python sim.py
```
